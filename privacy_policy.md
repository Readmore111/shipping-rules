隐私权政策

比利恩商品价格计算器

生效日期：2026年05月29日

本隐私权政策旨在说明 Chrome 扩展程序“比利恩商品价格计算器”（以下简称“本扩展程序”）在您使用过程中如何收集、使用、存储和共享您的数据。

1. 我们收集和处理的数据

本扩展程序在您使用过程中可能收集或处理以下类型的数据：

用户输入和用户活动数据：
包括您在商品价格计算、利润核算、面单生成、PDF/Excel 导出、发货查询等功能中主动输入的数据，例如商品采购价、重量、尺寸、售价、汇率、国家、渠道、包裹号、商品名称、收发件信息、备注等。这些数据用于执行对应功能。

网站内容：
当您访问店小秘批量编辑页面或速卖通商品页面时，本扩展程序会读取与功能相关的页面内容，例如商品标题、SKU、价格、运费、重量、尺寸、类目、国家/地区和商品页面展示信息，用于价格分析、类目识别、利润计算和标题生成。

账号和身份验证信息：
本扩展程序包含登录和授权功能，可能处理您输入的账号、密码、设备 ID、登录状态、授权状态、到期时间和最后登录时间。这些数据用于验证您是否有权限使用本扩展程序及相关功能。

本地设置数据：
本扩展程序会保存您的功能设置，例如 AI 服务商选择、模型设置、自定义接口地址，以及您自行配置的 API Key。这些数据保存在您的浏览器本地存储中，用于下次继续使用。

Cookie 和区域数据：
为计算速卖通不同国家或地区的商品价格和运费，本扩展程序可能读取或临时设置与速卖通区域、币种、站点请求相关的 Cookie。该处理仅用于实现价格、运费和地区相关功能。

公共配置数据：
本扩展程序可能从公开 URL 获取运费规则、商品类目数据或 AI 提示词模板等配置数据。这些数据是公开可访问的，用于支持利润核算、类目识别和标题生成功能。

2. 我们如何使用您的数据

我们处理上述数据主要用于以下目的：

提供核心功能：
用于商品价格计算、利润核算、运费计算、SKU 分析、类目识别、面单生成、PDF/Excel 导出、发货状态查询等功能。

登录和授权校验：
用于确认用户账号状态、设备授权状态、使用权限和到期状态。

AI 标题生成：
当您主动使用 AI 标题生成功能时，本扩展程序会将商品标题、提示词和相关请求内容发送给您选择的 AI 服务商或您配置的自定义 API 服务，用于生成或优化商品标题。

保存本地设置：
用于保存您的登录状态、插件设置、AI 服务商选择、API Key 和部分表单内容，方便您下次打开时继续使用。

获取和更新公共规则：
用于从公开地址获取最新的运费规则、类目数据或提示词模板，以保证计算和生成结果尽可能准确。

3. 数据存储

本扩展程序会将部分数据存储在您的浏览器本地存储中，包括登录会话、设备 ID、插件设置、AI API Key、计算表单数据或临时业务数据。

这些本地数据仅保存在您的设备浏览器中。您可以通过退出登录、清除浏览器数据、删除扩展程序或清空相关设置来删除这些本地数据。

账号授权、设备绑定、登录状态和部分发货/业务记录可能由开发者通过飞书多维表格进行维护，用于账号授权、设备管理和业务查询。

4. 数据共享

我们不会出售您的用户数据，也不会将用户数据用于广告投放、用户画像或与本扩展程序功能无关的追踪。

为实现本扩展程序的功能，部分数据可能会在必要时发送至以下服务：

飞书开放平台：
用于账号登录、设备授权、用户状态校验、发货状态查询或相关业务记录维护。

用户选择的 AI 服务商：
包括 MiniMax、DeepSeek、OpenAI、Anthropic Claude、Google Gemini 或用户自定义 API。只有当您配置 API Key 并主动使用 AI 功能时，相关标题、提示词和请求内容才会发送给对应服务商。

速卖通及其相关接口：
用于获取商品详情、价格、运费、SKU、类目和地区相关信息。

公共资源服务：
本扩展程序可能访问 GitHub、阿里云 OSS 或其他公开资源地址，以获取公开的规则、类目或提示词模板。

除上述为实现功能所必需的场景外，我们不会主动向其他第三方共享您的数据。

5. 关于 API Key 和访问凭证

本扩展程序允许用户自行配置 AI 服务商 API Key。API Key 会保存在您的浏览器本地存储中，并仅在调用您选择的 AI 服务商接口时使用。

请您妥善保管自己的 API Key、账号和密码，不要在不可信设备上保存敏感配置。

6. 数据安全

本扩展程序通过 HTTPS 与外部服务通信。我们会尽合理努力保护用户数据安全，并将数据处理限制在实现本扩展程序功能所必需的范围内。

7. 用户选择和控制

您可以选择不配置 AI API Key；未配置时，AI 相关功能将无法正常使用。

您可以在扩展程序设置中修改或清除 API Key、模型和自定义接口配置。

您可以通过退出登录、清除浏览器数据或卸载本扩展程序来删除本地存储的数据。

如需删除或更正飞书中维护的账号授权数据或业务记录，请通过本政策中的联系方式联系我们。

8. Chrome Web Store 用户数据政策

本扩展程序对用户数据的使用遵守 Chrome Web Store 用户数据政策，包括 Limited Use 要求。我们仅将用户数据用于提供和改进本扩展程序明确说明的功能，不会出售用户数据，也不会将用户数据用于与本扩展程序功能无关的广告、信用评估或转售。

9. 本政策的变更

我们可能会不时更新本隐私权政策。任何更改都将通过更新本页面上的“生效日期”来通知您。建议您定期查看本隐私权政策以了解任何更改。
